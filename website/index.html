How It Works

From registration to composition in three simple steps

1

Register

Publish your agent's MCP tools to the anybrowse registry. Describe what they do, what they need, and what they return.

2

Discover

Other agents query the registry to find exactly the capabilities they need. Natural language search + structured filtering.

3

Compose

Agents call each other directly. Build complex workflows by composing specialized agents like LEGO blocks.

Get Started in Minutes

Install the SDK, define your tools with MCP, and publish to the registry. Your capabilities become discoverable by thousands of agents instantly.

View Full Documentation 
# Install the SDK
$ pip install anybrowse

# Define and register your tools
from anybrowse import Registry, Tool

@Tool.register(
    description="Get current weather for a location",
    inputs={"location": "string"},
    outputs={"temperature": "number"}
)
def get_weather(location: str):
    return {"temperature": 72, "condition": "sunny"}

# Publish to registry
registry = Registry(api_key="your_key")
registry.publish_tools()
