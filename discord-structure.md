## SERVER RULES

1. **Be respectful** — Treat everyone with respect. Healthy debate is encouraged; harassment is not.

2. **No spam/self-promo** — Share your projects in #showcase. Don't dump links without context.

3. **Stay on topic** — Use the right channel for your discussion.

4. **No crypto/NFT shilling** — This is an infrastructure project, not a token scheme.

5. **Help others** — The best way to learn is to teach. Answer questions when you can.

6. **Credit your sources** — When sharing code or research, give credit where it's due.

---

## MODERATION GUIDELINES

- Use timeouts for minor infractions (cool-off periods)
- Use kicks for repeated violations after warning
- Use bans for spam, harassment, or malicious behavior
- All moderators should document actions in private mod channel
