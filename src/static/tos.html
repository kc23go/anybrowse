1. What anybrowse does

anybrowse is a web scraping API operated by Heimat LLC, operating as anybrowse, that converts any URL to clean, LLM-ready Markdown. We use real Chrome browsers to render pages (including JavaScript-heavy sites) and return the content as Markdown text. The API is available at anybrowse.dev.

Our endpoints include:

  • /scrape — convert a single URL to Markdown
  • /crawl — follow links and convert multiple pages
  • /serp/search — run a Google search and return Markdown results
  • /mcp — Model Context Protocol server for AI agents
