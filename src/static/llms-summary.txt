## API Endpoints

### POST /scrape
Extract markdown, raw HTML, optional screenshot (base64 PNG), and links from a single URL. $0.002/call.

**New fields (Firecrawl-compatible):**
- `html` — raw page HTML (always returned)
- `screenshot` — base64 PNG of viewport (set `"screenshot": true` in request)
- `links` — array of all hrefs (set `"includeLinks": true` in request, max 200)

### POST /crawl
Search Google and scrape top results. $0.01/call.

### POST /serp/search
Real-time web search via Brave Search API. Returns titles, URLs, and snippets as structured JSON. $0.002/call.

### POST /batch
Scrape up to 10 URLs in parallel, single request.
```json
{ "urls": ["https://site1.com", "https://site2.com"] }
```
Free: 5 batches/day. Pro: unlimited.

### POST /watch
Monitor a URL for content changes, get webhook alerts.
```json
{ "url": "https://example.com", "webhook_url": "https://your-server.com/hook", "interval_minutes": 60 }
```
GET /watches — list your watches. DELETE /watch/:id — remove a watch.

### POST /extract
Extract structured data from any URL as JSON. $0.01/call.
```json
{ "url": "https://shop.example.com/product", "schema": {"price": "number", "title": "string", "in_stock": "boolean"} }
```

### GET /mcp
MCP server: scrape, crawl, search, batch_scrape, extract tools.

## x402 Micropayments (AI agents with crypto wallets)

Pay per-request with USDC on Base — no subscription needed.

- Use `wrapFetch` from the `x402` npm package: `npm install x402`
- Or use Coinbase AgentKit — x402 is handled automatically
- $0.002 USDC per /scrape, $0.01 per /crawl
- The API returns HTTP 402 with payment requirements when you hit the free tier limit

**Common error:** "paymentPayload is invalid" — always use the x402 client library. Do NOT manually construct the X-PAYMENT header.
**Docs:** https://anybrowse.dev/docs/x402

```javascript
import { wrapFetch } from 'x402/client';
const fetch402 = wrapFetch(fetch, wallet);
const response = await fetch402('https://anybrowse.dev/scrape', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' })
});
```
