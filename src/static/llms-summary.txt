# AnyBrowse - Web Scraping for AI Agents

Web scraping → Markdown extraction for AI agents. 10 free scrapes/day. No signup required for free tier.

## What It Does

Converts any webpage into clean, structured markdown for LLM processing. Built specifically for AI agents and RAG pipelines.

## Key Features

- **90%+ success rate** with stealth browser mode (anti-bot bypass)
- **Proxy fallback** — automatically retries blocked sites via residential proxies
- **30s timeout** for JS-heavy single-page applications
- **Structured error responses** with machine-readable reason codes (`TIMEOUT`, `BLOCKED`, `NOT_FOUND`, etc.)
- Real Chrome browser rendering (not headless scripts)
- MCP server for Claude/Cursor/Windsurf
- Clean markdown output (removes ads, navigation)
- Flexible authentication: free tier, API keys, OR wallet-based

## API Endpoints

### POST /scrape
Extract markdown from single URL.
```json
{
  "url": "https://example.com",
  "render": true,
  "waitFor": 2000
}
```

### POST /crawl
Crawl multiple URLs from starting point.
```json
{
  "url": "https://example.com/docs",
  "maxPages": 10,
  "maxDepth": 2
}
```

### GET /mcp
MCP server endpoint for AI tool integration.

## MCP Integration

Add to Claude/Cursor/Windsurf config:
```json
{
  "mcpServers": {
    "anybrowse": {
      "command": "npx",
      "args": ["-y", "@anybrowse/mcp-server"]
    }
  }
}
```
